# Hi, I'm Stephen Pearse

**Audio/AI Engineer** — I build production software at the intersection of music technology, real-time systems, and applied AI.

13+ years spanning academic research, technical leadership, and AI system design. PhD in music computing. Currently focused on LLM-powered music tools and agentic systems.

---

## What I work on

**Agentic AI systems** — multi-agent pipelines with LangGraph, deployed on LangGraph Platform. Interrupt-aware, streaming, production-grade. Most recently: a voice-driven music production assistant where a graph of specialised agents handles intent routing, palette curation, arrangement, and effects end-to-end.

**Real-time audio software** — C++/JUCE desktop applications with <10ms latency, hardware integration, and DSP. Cross-platform (macOS + Windows). CI/CD, crash monitoring, release automation. Scaled to 2,500 daily active users at festivals worldwide.

**ML infrastructure** — audio embedding pipelines (MERT → CLaMP3), vector search (FAISS, Google Vertex AI), batch processing optimisation. Built production-ready from scratch in weeks, not months.

**Full-stack** — TypeScript, NestJS, React, PostgreSQL. Financial systems, webhook idempotency, payment integrations.
